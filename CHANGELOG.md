# Changelog

## v0.1.0 - 2026-07-11

The first public ArtistPass release.

### Included

- Cinematic fictional actor portfolio with showreel, headshots, role-fit dossier, resume and casting-card flows.
- Browser Admin for editing the profile, previewing changes and publishing live.
- Vercel Blob storage for shared config, image and PDF updates.
- One-click Vercel deployment for developers and an assisted setup path for non-technical artists.
- Vercel Web Analytics and a completed-deployment return page.
- Mobile layout, sharing flows, downloads, SEO basics and a read-only public demo.
- Public screenshots, build stories and open-source contribution guidance.

### Current Limits

- The self-serve deployment path still expects a Git provider and Vercel account.
- Video uploads are not included; artists paste YouTube, Google Drive, Vimeo or other hosted links.
- The public profile is fictional sample material and should be replaced before a real artist publishes a copy.
