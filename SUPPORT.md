# ArtistPass Support

## Using Or Deploying The Template

Start with the [README](README.md) and [one-click deploy guide](docs/ONE_CLICK_DEPLOY.md).

## Found A Bug?

Open a bug report and include the page, browser, steps to reproduce and any safe screenshots or error messages. Never post passwords, tokens or private artist material.

## Have An Idea?

Open a feature request. Explain who the change helps and what they are trying to do. A small example is more useful than a long feature list.

## Have A Question?

Ask in [GitHub Discussions](https://github.com/eyeinthesky6/artistpass/discussions). Use an issue only when you can describe a reproducible problem or focused improvement.
