## LinkedIn

I kept seeing artist portfolios split across Instagram, Drive folders, YouTube links, PDFs and old website pages.

So I built ArtistPass: a small open-source portfolio website for actors and other performers.

It brings the useful pieces into one place:

- showreel and role-specific clips;
- headshots and gallery;
- resume and casting card;
- contact and sharing flows;
- a browser Admin for routine updates.

The interesting constraint was not just making the page look polished. It was making updates possible without handing an artist a codebase or a heavy CMS.

Developers can clone and deploy the template through Vercel. Non-technical artists can use an assisted setup path and then manage normal text, image and PDF updates from the website Admin.

The demo uses a fictional artist, and the code is available under the MIT License.

This is an experiment, not a traction announcement. I am putting the working product in the open and seeing where it is genuinely useful.

Live demo: https://artistpass.vercel.app

GitHub: https://github.com/eyeinthesky6/artistpass

## Short Personal Note

I made ArtistPass because the existing options often felt either visually generic, too expensive for a simple portfolio, or too complicated to maintain.

The result is intentionally small: one strong artist website, a browser Admin, an open-source developer route and an assisted path for people who never want to see GitHub.

It may stay a useful little open-source project. It may become something larger if artists actually pull it in that direction. For now, the right move is to release it and learn.
