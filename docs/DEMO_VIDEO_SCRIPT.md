# 90-Second Demo Video Script

Use this to record the first demo for LinkedIn, Product Hunt, README and outreach.

## Goal

Show that ArtistPass is not just a pretty profile. It is a self-managed portfolio website that can be updated from the browser.

## Structure

### 0-10 sec: Problem

Voiceover:

> Actors and artists usually share scattered links: photos in one place, reels in another, resume somewhere else. ArtistPass turns that into one polished, share-ready profile.

Screen:

- Open `https://artistpass.vercel.app`.
- Show hero, reel, role-fit cards.

### 10-30 sec: Public Site

Voiceover:

> The profile has a cinematic first screen, role-fit sections, reel links, headshots, a casting card, resume download and contact/share buttons.

Screen:

- Scroll through hero, role fit, reel, headshots, pass-it-on section.

### 30-55 sec: Admin

Voiceover:

> The artist does not need a full CMS. Admin opens section by section. They can update text, links, images, PDFs and favicon, then preview locally.

Screen:

- Open footer Admin.
- Show Hero, Reel/portfolio and Contact sections.
- Do not show real password.

### 55-75 sec: Publish

Voiceover:

> When ready, Publish live writes the update to Vercel Blob. Everyone sees the change after refresh. No developer is needed for routine updates.

Screen:

- Show Publish live button.
- Show live site refresh.

### 75-90 sec: CTA

Voiceover:

> Developers can use the open-source template. Artists can ask for assisted setup. The first pilot setups are available now.

Screen:

- Show README Make It Yours button.
- Show footer Assisted setup button.

## Caption

I built ArtistPass: a self-managed actor and artist portfolio website template.

It gives performers one polished link with:

- showreel/reel links;
- headshots;
- casting card;
- resume/CV;
- native share/contact flows;
- browser Admin publishing.

Open-source template for makers. Assisted setup available for artists who do not want to touch GitHub/Vercel.

Live demo: https://artistpass.vercel.app
Repo: https://github.com/eyeinthesky6/artistpass
